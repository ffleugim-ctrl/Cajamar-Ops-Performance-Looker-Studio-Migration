# Cajamar-Ops-Performance-Looker-Studio-Migration
Este projeto apresenta a migração de um fluxo de relatórios operacionais para o Looker Studio, focado na unidade logística de Cajamar (SP). O objetivo principal é fornecer uma visão 360° da performance de SLA e produtividade, permitindo que o RH (Strategic People Analytics) identifique gargalos de pessoal em tempo real.
